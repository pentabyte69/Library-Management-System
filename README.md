# Library-Management-System
Author: Mohamad Suhail  
Course: CS Final Project  

This project is a C++ Library Management System that allows a user to:

- Add books
- Search books
- Borrow books
- Return books
- Manage patrons
- Track transactions

The system demonstrates object-oriented programming concepts such as:
- Classes
- Inheritance
- File I/O
- Encapsulation
